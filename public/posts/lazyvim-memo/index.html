Lazyvim-Memo

I come across an error when I tried to use nvim in iterm2(set to a background). The down side of termnial start to wink when using nvim. The error is fixed by changing the nvim background to transparent.

add a new file in ~/.config/nvim/lua/plugins/colorscheme.lua

return {
{
  "folke/tokyonight.nvim",
  opts = {
    transparent = true,
    styles = {
      sidebars = "transparent",
      floats = "transparent",
    },
  },
},
}

Note:

Actually, it doesn’t work at all. Fixed by dragging the little bar down side to the bottom.
